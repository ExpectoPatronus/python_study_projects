Интерфейс имеет следующие компоненты:

1. Поле ввода: здесь пользователь может вводить выражение, которое нужно вычислить.
Это может быть простое арифметическое выражение или тригонометрическое выражение.
Поле ввода не может принимать ввод с клавиатуры для ввода выражения необходимо пользоваться кнопками на калькуляторе.
Ввести возможно строку до 255 символов.

2. Кнопка "=": эта кнопка инициирует вычисление выражения, введенного в поле ввода.
Когда пользователь нажимает кнопку "=", программа должна разобрать выражение, вычислить его и отобразить результат.

3. Кнопка "AC": эта кнопка очищает поле ввода и область отображения результата.

4. Кнопка "⌫": эта кнопка очищает поле ввода по одному символу

5. Область отображения результатов: в этой области отображается вычисленный результат введенного в поле ввода выражения.
Она обновляется автоматически при нажатии пользователем кнопок на калькуляторе.

6. Кнопка "Graph": эта кнопка позволяет пользователю нарисовать график выражения, введенного в поле ввода.
Когда пользователь нажимает кнопку "График", программа должна построить график выражения и отобразить его в отдельном окне.
Окно с графиком содержит меню, которое позволяет изменять масштабы графика, приближать или двигаться по графику.
Область определения и область значения функций ограничиваются по числами от -1000000 до 1000000

7. Кнопка "Справка": эта кнопка предоставляет помощь и инструкции по использованию калькулятора.
Когда пользователь нажимает кнопку "Справка", появляется всплывающее окно с инструкциями по использованию калькулятора.

8. Поле "history": Хранит историю операций, позволять загружать выражения из истории и очищать историю целиком.

9. Кнопка "Clean history": Очищает историю целиком из поля "history".

10. Кнопка "Load string": Загружает выбранное выражение из поля "history".
